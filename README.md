# 📖 Blogiverse <a name="about-project">

Hi there! Welcome to Blogiverse</a>

Blogiverse is a website where users may write and publish blog articles. It offers a platform for bloggers to share their views and ideas with the rest of the world, as well as for readers to find new material, participate in discussions, and like blog posts.

## 🛠 Built With <a name="built-with"></a>

- **Like blogs**
- **Comment on blogs**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to install the following:

- Ruby
- PostgreSQL
- Rails gem

### Setup

Clone this repository to your desired folder:

```snippet
  git clone https://github.com/UbdaNam/Blogiverse.git
```

### Install

```snippet
  bundle install
```

Install this project with: Any code editor or terminal which you are comfortable with

### Usage

In the project directory, you can run:

```Database
  psql -U [enter your username here] -h localhost
  password:_"Enter your password"
  CREATE DATABASE blogiverse;
```

```Migrate db
rails db:migrate
```
